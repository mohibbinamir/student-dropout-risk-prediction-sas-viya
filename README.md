# Student Dropout Risk Prediction Using SAS Viya

This repository contains a **group predictive modelling project** developed in **SAS Viya Model Studio** for predicting student dropout risk.

## Project Summary

The project used the `STUDENT_DROPOUT_FINAL` dataset to classify students into:

- `Dropout`
- `Not_Dropout`

The final dataset contained **4,424 observations** and **39 variables**, and the work followed the **CRISP-DM** methodology.

## Models Built

The group developed and compared six predictive models:

- Neural Network
- Support Vector Machine (SVM)
- Decision Tree
- Forest
- Logistic Regression
- Gradient Boosting

## My Contribution

This was a **group assignment**. My individual contribution focused on:

- **Neural Network**
- **Support Vector Machine (SVM)**

I worked on model construction, tuning, validation, and interpretation for these two models.

## Key Results

- The tuned SVM achieved the best validation misclassification among my two models.
- The **Neural Network** was selected as the **project champion model** by SAS Viya based on the KS statistic.
